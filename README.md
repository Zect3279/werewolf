## コマンド
- `/start` => ゲーム開始
- `/join` => ゲームへの参加
- `/vote <@role[discord.Role]>` => 追放者への投票
- `/raid <@role[discord.Role]>` => 殺害者を指定（人狼専用コマンド）
- `/fortun <@role[discord.Role]>`=> 占い対象を指定（占い師専用コマンド）
